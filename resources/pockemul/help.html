Pocket interface

Use the pocket as in real life. Power it on by the power interrupter, reset it by the reset hole. If the key is not visible on the front, rotate the pocket by a swipe in the desired direction.

When you power off or close the pocket, PockEmul automatically save the context.

To connect a pocket to an extension, a printer for example, simply clic on the connector to show compatible extensions. You will need to rotate the pocket to show the connector.

If the clic on the connector is not yet managed, start the extension by the main menu.

To connect the extension, simply position the relatives connectors side by side and pockemul will automatically make the link. If not, define the link manualy by the advanced context menu.

Context menu

Double tap on the pocket (not on a key) to show the context menu.

Context menu entries:

  • Maximize
  • Minimize
  • Enable pinch on the pocket (leave this mode by double tap)
  • Advanced context menu

Cloud

Activity tracking

PockEmul send activity events to Google Analytics server. You can consult GA Privacy Policy

The main goal is to identify which function is used and which pocket is often emulated.

All information sent to GA are anonymous, and the IP is anonymous too. At the first start, PockEmul ask the user to enable or disable tracking system. The tracking enabled switch is accessible in the settings screen.

Each event is completed by those data:

  • PockEmul ID : a unique Hash code created at the first startup. You can change it in the config.ini file located in your <home directory/pockemul> (just delete the line, il will be regenerated at startup)
  • Operating System
  • Screen resolution
  • Geographic location : Country and town
