PockEmul
========
PockEmul is a Multi-Pocket Computer emulator.

![](http://pockemul.free.fr/forum/dev/device1.png)

It is develop in C++ using Qt4 or Qt5.

It is available on Win32, Linux, Mac OS X and Android

http://pockemul.free.fr
